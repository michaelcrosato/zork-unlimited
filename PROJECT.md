# Project: zork-unlimited

## Architecture
- `src/`: Core engine logic, CYOA mechanics, parsers, and RPG modules.
- `ui/`: User interface components and assets.
- `tests/`: Extensive unit and integration test suites, including regression and property testing.
- `content/`: CYOA adventure and RPG content packs.
- `scripts/`: Utilities for integrity verification, playtesting, and loops.
